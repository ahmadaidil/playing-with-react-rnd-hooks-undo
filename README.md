# playing-with-react-rnd-hooks-undo

## Dependencies
* nextjs
* react, with new api: hooks
* use-undo: undo/redo functionality with React Hooks.
* react-rnd: draggable & resizable component.

## How to use

Install it and run:

```bash
npm install
npm run dev
# or
yarn
yarn dev
```
